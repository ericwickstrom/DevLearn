# DevLearn

A mobile-friendly flashcard web app for technical interview preparation.

## Features

- 20+ technical interview questions covering:
	- Data Structures & Algorithms
	- Big O Notation
	- System Design
	- JavaScript & React
	- Databases
	- Web Fundamentals
	- Coding Patterns
- Flip cards to reveal answers
- Navigate between questions
- Responsive mobile design
- Dark theme for comfortable studying

## Getting Started

### Prerequisites

- Node.js 20+ (LTS)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

## Deployment

This app is configured to deploy automatically to GitHub Pages via GitHub Actions.

1. Push code to your repository
2. Enable GitHub Pages in Settings → Pages → Source: GitHub Actions
3. The app will deploy automatically on push to `main`

## Tech Stack

- React 18
- Vite
- CSS3 (Flexbox, CSS animations)
